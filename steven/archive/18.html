Microsoft Anti-Cross Site Scripting Library V1.5.

Microsoft's ACE Team, announced the release of the Anti-Cross Site Scripting (XSS) Library version 1.5.

Besides the HtmlEncode and UrlEncode methods that were available in the 1.0 release, the 1.5 release also has XmlAttributeEncodeXml, EncodeVisualBasicScriptEncode, EncodeJavaScriptEncode and HtmlAttributeEncode methods. Besides these improvements the Library has now an EULA that provides the ability to deploy into production environments, which 1.0 didn't.

You can find more information on the ACE Team blog and download the library over here.
